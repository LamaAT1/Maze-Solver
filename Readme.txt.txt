 Maze Solver Game

A C++ application that reads a text‐based maze, finds a path from **S** to **E**, and marks it with `*`.  
Supports both recursive DFS and iterative BFS.

---

Features

- **Algorithms**:  
  - **DFS** (recursive backtracking)  
  - **BFS** (iterative, level‐by‐level)  
- **Save/Load**: Read/write mazes as plain `.txt`.  
- **CLI**: No dependencies beyond C++11.
